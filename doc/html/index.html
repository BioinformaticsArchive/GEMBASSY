GEMBASSY

The GEMBASSY package is intended to achieve higher usability of the G-language SOAP service.

Applications in the current gembassy release

Program name Description
gaminoinfo Prints out basic amino acid sequence statistics
gbaseinfocontent Calculates and graphs the sequence conservation using information content
gbui Calculates base usage indices for protein-coding sequences
gcai Calculate codon adaptation index (CAI) for each gene
gcbi Calculates the codon bias index (CBI)
gcgr Create a Chaos Game Representation of a given sequence
gfindoriter Predict the replication origin and terminus in bacterial genomes
ggcsi GC Skew Index: an index for strand-specific mutational bias
ggcskew Calculate the GC skew of the given genome
ggcwin Calculate the GC content along the given genome
ggeneskew Calculate the gene strand bias of the given genome
ggenomicskew Calculate the GC skew in different regions of the given genome
gicdi Calculates the intrinsic codon deviation index (ICDI)
gphx Identify predicted highly expressed (PHX) genes
gqueryarm Get the replication arm name (left or right) from the given position
gquerystrand Get the strand name (leading or lagging) from the given position
greporiter Get the positions of replication origin and terminus
gseq2png Converts a sequence to PNG image
gseqinfo Prints out basic nucleotide sequence statistics
gshuffleseq Create randomized sequence with conserved k-mer composition
