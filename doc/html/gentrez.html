gentrez

 

Function

Search NCBI Entrez in G-language Shell

Description

Search NCBI Entrez with keyword through EUtilities. This is intended for quick lookup through G-language Shell, so only top ten hits are reported.

G-language SOAP service is provided by the Institute for Advanced Biosciences, Keio University.
The original web service is located at the following URL:
http://www.g-language.org/wiki/soap

WSDL(RPC/Encoded) file is located at:
http://soap.g-language.org/g-language.wsdl

Usage


   Here is a sample session with gentrez

% gentrez NC_000913.gbk
(null)

gentrez reads nucleotide sequences.

Ouptput file format

The output from gentrez is to standard output.
